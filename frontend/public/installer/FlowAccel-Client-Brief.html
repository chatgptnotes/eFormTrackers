Getting Started Brief

What you're receiving & what we need from you

Prepared for: Client / IT Administrator  |  A one-page brief to read before opening the installer
FlowAccel is a workflow management application that runs entirely on your own server — your forms, approvals, signatures and data never leave your premises. You'll receive two items: (1) the Windows installer, and (2) a short PDF asking your IT team for a few details.

What your IT team needs to do — in order

1

Prepare a Windows server

  • Windows 10 / Server 2016 or newer
  • 10 GB free disk space on C:
  • Administrator access
  • A fixed internal IP address
  • No internet needed — everything's bundled
2

Fill the Azure AD info sheet

  • Register FlowAccel inside Microsoft Entra ID
  • Send us App ID, Tenant ID, Client Secret
  • Lets staff sign in with Microsoft 365
  • Client Secret must be shared securely
3

Install 7-Zip on the server

  • Free utility — 30 seconds to install
  • From https://www.7-zip.org
  • Needed to extract the installer parts
4

Run the installer

  • Download 5 parts + open .7z.001 with 7-Zip
  • Right-click the .exe → Run as Administrator
  • Walk through 6 wizard pages (IP, passwords, cert)
  • Wait 10–15 minutes for install to finish

What happens after installation

Auto-starts with the serverRuns as a Windows service. Survives reboots without manual intervention.
Access on your networkUsers open https://<server-ip>/ from any browser inside your LAN.
Local databaseAll workflow data stored in a PostgreSQL database on the same server.

Three things to keep in mind

  • All on-premises, no cloud dependency. Forms, approvals, and signatures stay on your hardware.
  • One-time setup, then hands-off. The installer is idempotent — safe to re-run if anything goes wrong.
  • Microsoft sign-in is optional but recommended. Saves managing a separate password for each user.
Security note: The Client Secret (from the Azure AD form) and any certificate private key are highly sensitive. Send via a password-manager share, encrypted file, or secure upload link — never plain email. The App ID and Tenant ID are not secret.
Before we kick off — please confirm:
  • Which Windows server you'll install on
  • Completed Azure AD information sheet returned
  • 7-Zip installed on the target server
  • A 30-minute window booked for your IT person to run the wizard (we'll be on-call)
